# instago
Instagram Backend API clone, in Golang

## API Reference

#### Create users 
```http
  POST /users
```

#### Get users By ID 

```http
  GET /users/{id}
```
#### Create posts 

```http
  POST /posts
```
#### Get post By ID 

```http
  GET /posts/{id}
```
#### Get all post by a user

```http
  GET /posts/users/{id}
```
